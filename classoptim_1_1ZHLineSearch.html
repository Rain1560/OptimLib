Detailed Description

template<typename fp_t = double, bool use_prox = false>
class optim::ZHLineSearch< fp_t, use_prox >

Zhang-Hager line search.

find a step [\alpha_k] such that \( f(x_k + \alpha_k d) \leq C_k + \rho \alpha_k \nabla f(x_k)^T d \), where \( C_k = \frac{\gamma Q_{k-1}C_{k-1} + f(x_k)}{Q_k} \), \( Q_k = \gamma Q_{k-1} + 1,\ C_0 = f(x_0) \).
